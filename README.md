📌 Overview
AI Resume Analyzer is an AI-powered web application developed to help job seekers evaluate their resumes against specific job descriptions. The application analyzes resumes, estimates ATS compatibility, and provides AI-generated recommendations to improve resume quality and increase interview opportunities.

✨ Features
📄 Upload PDF Resume

🤖 AI-based Resume Evaluation

📊 ATS Compatibility Score

🎯 Resume & Job Description Matching

💡 Personalized Resume Improvement Suggestions

📥 Download Analysis Report

🛠️ Tech Stack
Python

Streamlit

Google Gemini API

PyPDF2

FPDF

Python-dotenv

⚙️ Installation
git clone <repository-url>
cd AI-Resume-Analyzer
Create a virtual environment:

python -m venv venv
Activate it:

Windows

venv\Scripts\activate
Install dependencies:

pip install -r requirements.txt
Create a .env file:

GOOGLE_API_KEY=YOUR_GOOGLE_API_KEY
Run the application:

streamlit run app.py
Visit:

http://localhost:8501
🚀 How to Use
Upload your resume in PDF format.

Enter the job description.

Click Analyze Resume.

Review the ATS score and AI-generated feedback.

Download the generated report.
